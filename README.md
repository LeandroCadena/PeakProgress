# PeakProgress

PeakProgress is a mobile fitness application focused on workout tracking, progress monitoring, and habit building.

## Features

- User authentication
- Workout routines
- Exercise tracking
- Rest timer
- Progress analytics
- Personal records
- Cloud synchronization

## Tech Stack

- React Native
- Expo
- TypeScript
- Supabase
- PostgreSQL
- Zustand
- React Navigation

🚧 Currently under active development.
